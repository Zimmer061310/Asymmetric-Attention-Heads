# Configs

This directory contains YAML configs for AAH-v3 paper runs, diagnostic sweeps,
and earlier exploratory variants.

Paper-facing config families:

- `paper_required/`: main and stress-test configs from the paper experiment plan.
- `paper_final_512/`: earlier 512-token paper config set.
- `aah_v3_*`: AAH-v3 development, ablation, and scale configs.

Older AAH-v1/AAH-v2 configs are retained for traceability, but the final paper
method is AAH-v3.
