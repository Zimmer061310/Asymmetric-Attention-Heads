# AAH FLOPs Reduction Lab Results

This directory is reserved for compact copyout summaries from
`experiments/aah_flops_reduction_lab`.

Allowed here:

- summary CSV/MD files;
- compact `gpu_flops_profile.json` outputs;
- static plan JSONs needed to reproduce a profile;
- small manifests and run logs.

Do not commit checkpoints, raw W&B run directories, raw Nsight reports, or
server credentials.
